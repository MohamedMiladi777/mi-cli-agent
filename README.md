
## Overview

The last year was the year of agents. While agents are powerful tools to automate mundane tasks, students and Junior devs are skipping
coding fundamentals, which affect their learning skills, cognitive engagement and damages the gist of engineering. 

This agent attempts to solve this problem. Insread of  auto completing absolutely everything, the agent will act as a pair programmer
by auto completing real mundane tasks and leave the problem solving tasks to the developer.



## How to run the agent


First build the project using :
``` npm run build ```

and then run ```mi``` command.
